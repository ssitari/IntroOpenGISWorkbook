# Introduction to GIS Using Open Source Software

A self-directed introductory workshop text for learning geographic 
information systems (GIS) using open source tools, with a focus on 
QGIS 3.44 and New York City as the primary geographic context. 

Designed for researchers, graduate students, and faculty across 
disciplines who want to incorporate spatial methods into their work. 
No prior GIS experience assumed.

## Status

🚧 **Work in progress** — This text is actively being developed 
during summer 2026. Content is incomplete and subject to change. 
A first complete version is expected in September 2026.

## Contents

- Introduction: What GIS is and what it's for
- Chapter 1: Exploring the QGIS Interface
- Chapter 2: Working with Data
- Chapter 3: Geographic Analysis
- Chapter 4: Thematic Mapping and Cartographic Output
- Chapter 5: Creating Spatial Data (Georeferencing and Digitizing)
- Chapter 6: Going Further

## Data

Workshop data is provided as GeoPackage files and can be downloaded 
from this repository. Instructions for accessing data from original 
sources are included in each chapter.
